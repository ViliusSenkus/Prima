slice(), substring(), substr()

Skirtas išimti iš string kintamojo reikšmės simbolius nurodant norimų išimti simbolių eilės numerius

Naudojimas
var.slice(x,y), jeigu reikšmės neigiamos, eilės numeris skaičiuoti pradedamas iš galo
var.substring(x,y),
var.substr(first symbol, number od symbols)
x - pirmo imamamo simbolio eilės numeris, y - simbolio, iki kurio išimamams tekstas, eilės numeris, z - imamų simbolių skaičius.

Kodas:

<p>iš kintamojo "blabla" skirtingais būdais išimti žodžiai "išimk" "man" "ždožius" :<span id="isimk"></span></p> <button onClick="take()">išimk žodžius</button> <script> function take(){ blabla = "išimk iš manęs žodžius"; pirmasZodis = blabla.slice(0, 5); antrasZodis = blabla.substring(9,12); treciasZodis = blabla.substr(15,7); document.getElementById("isimk").innerHTML = " " + pirmasZodis + " " + antrasZodis + " " + treciasZodis; } </script>

replace(), replaceAll()

Skirtas pakeisti kintamojo reikšmėje esantį žodį į naują

Naudojimas
var.replace("senas žodis", "naujas žodis") - pakeičia pirmą sutiktą seną žodį į naują žodį, -
var.replace(/SENAS ŽODIS/i, "naujas žodis") - keičia žodžius nepriklausomai nuo seno žodžio raidžių dydžio,
var.replace(/senas žodis/g, "naujas žodis") - pakeičia visus sutiktus senus žodžius. var.replace()

Kodas:

<p>iš kintamajame "text" įdėtos reikšmės "Būdavo būdavo iš mūsų šilelio" skirtingais būdais pakeičiami žodžiai "būdavo" į "tebėra":<span id="keisk"></span></p> <button onClick="keis()">Parodyk pakeistą tekstą</button> <script> function keis(){ let text = "Būdavo būdavo iš mūsų šilelio, būdavo ir ne taip būdavo"; let text1 = text.replace("būdavo","tebėra"); let text2 = text.replace(/būdavo/i,"tebėra"); let text3 = text.replace(/būdavo/gi,"tebėra"); let text4 = text.replaceAll("būdavo","tebėra"); document.getElementById("keisk").innerHTML="<br />"+text1+"<br />"+text2+"<br />"+text3+"<br />"+text4; } </script>

toUpperCase(), toLowerCase()

Skirtas pakeisti kintamojo reikšmės raides į didžiąsias arba mažąsias

Naudojimas
var.toUpperCase() - pakeičia reikšmę į didžiąsias raides, -
var.toLowerCase() - pakeičia reikšmę į mažąsias raides.

Kodas:

<p>kintamajame "text" reikšmė "Vilnius Kaunas ir Klaipėda" rodoma vien didžiosiomis ir vien mažosiomis raidėmis:<span id="ud"></span></p> <button onClick="chang()">Parodyk pakeistą tekstą</button> <script> function chand(){ let text = "Vilnius Kaunas ir Klaipėda"; let text1 = text.toUpperCase(); let text2 = text.toLocaleLowerCase(); document.getElementById("ud").innerHTML = "<br />dižiosios raidės: "+text1+"<br /> mažosios raidės: "+text2; } </script>

contact()

Skirtas sukurti naują kintamąjį sujungus kitų dviejų kintamųjų vertes

Naudojimas
1stVar.contact("a", 2ndVar, "a", 3rdVar)
a - bet koks simbolis turintis būti tarp kintamūjų (paprastai tarpas)

Kodas:

<p>Iš 2 kintam...unas ir Klaipėda"<span id="joi"></span></p> <button onClick="joigas()">Sujungtas tekstas</button> <script> function joigas(){ var text1 = "Vilnius"; var text2 = "Kaunas"; var text3 = text1.contact("", text2); document.getElementById("joi").innerHTML = text3 ; } </script>
